# Detector de Rostos com Haar Cascade

## Autores
- Luca Aguiar
- Igor Gabriel
- João Luiz
- Miguel Albuquerque

## Descrição
Aplicação em tempo real para detectar rostos utilizando classificadores Haar Cascade do OpenCV. O programa captura vídeo da webcam, identifica rostos e aplica um filtro criativo tipo "ciborgue" com olhos vermelhos e máscara escura na região da boca.

## Funcionalidades
- Detecção de rostos em tempo real via webcam
- Desenho de retângulo ao redor dos rostos detectados
- Efeito visual criativo com filtro ciborgue (olhos biônicos e máscara)
- Interface simples com tecla 'q' para sair

## Uso
Execute o script Python:
```bash
python detector_rostos.py
```

Pressione 'q' para encerrar a aplicação.

## Dependências
- OpenCV (`cv2`)
- Python 3
